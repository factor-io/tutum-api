Ruby API for Tutum
======================

This library implements [Tutum's API](https://docs.tutum.co/v2/api/). It was spawned from the [ruby-tutum](https://github.com/255BITS/ruby-tutum) gem.

## Installation

```
$ gem install tutum-api
```
##Configuration

Basic authorization:

```ruby
  require 'tutum'
  tutum = Tutum.new(TUTUM_USERNAME, API_KEY)
```
